# Otzoviki KZ Design Kit

## Product scene
A homeowner in Kazakhstan is checking a repair contractor on a phone before paying an advance. The UI must feel calmer and more trustworthy than a classifieds page, but more useful and direct than a methodology report.

## Primary SEO intent for homepage
- Primary key: `ремонтные компании Казахстан отзывы рейтинг`
- Supporting intents: `проверить ремонтную компанию`, `отзывы о ремонтных компаниях`, `рейтинг ремонтных компаний Казахстан`, `как выбрать подрядчика на ремонт`
- Homepage role: broad trust/directory entry. City/service pages carry local commercial keys like `ремонт квартир Алматы`.

## Market references used
Accessible/reviewed:
- Taskrabbit: short hero promise, one dominant task selector, category chips, proof stats below.
- Zoon Kazakhstan: search + city at the top, category chips, list cards with rating/address/call actions.
Blocked but considered as known category references, not copied directly: Houzz, Angi, Thumbtack, Yelp, 2GIS, Checkatrade, MyBuilder.

## Otzoviki positioning
Not a lead-selling marketplace. Not a pure reviews wall. Otzoviki is a pre-contract verification layer: reviews + rating + external sources + risk signals + right of reply.

## Visual direction
Name: **Warm Evidence Marketplace**
- Warm sand background for home/repair context.
- Deep green for trust and verified actions.
- Amber for caution/advance-payment decisions.
- Blue only for external evidence/source links.
- No purple gradients, no glassmorphism, no black/white extremes.

## Color tokens
- Background: `#f5f2ea`, warm sand.
- Surface: `#fffdfa`, paper.
- Ink: `#151716`, soft black.
- Trust green: `#153d2b`, `#246246`, `#eef5ef`.
- Amber decision: `#e4a53a`, `#fff2d8`.
- Evidence blue: `#2457a7`, `#eef4ff`.
- Risk red: `#a93a32`, `#fff0ee`.

## Typography
- Display: Literata, Georgia, serif. Use for big editorial trust headlines.
- Body/UI: Manrope, system-ui, sans-serif. Use for search, cards, tables, navigation.
- Mono/data: ui-monospace stack for internal/debug only.
- H1 should be short: 6-9 words, max 2-3 mobile lines.

## Homepage first-screen rules
1. One H1: action + outcome before payment.
2. One large search field above the fold.
3. 3-4 popular actions max.
4. Infographic/trust flow can explain the product visually, not as a text wall.
5. No methodology, no operator terms, no English labels above the fold.
6. Main homepage copy must naturally support the primary SEO intent without stuffing.

## Components
- Hero search card: large, rounded, white, with category chips underneath.
- Trust flow card: 4 visual nodes: отзывы, контакты, источники, риски.
- Action cards: numbered quick routes, short labels only.
- Company preview card: photo placeholder + company name + human description.
- Evidence/risk/right-of-reply cards: lower page, not first screen.

## Copy rules
Avoid:
- “Путь как у нормального marketplace”
- “Guide checklist”
- “External footprint”
- “verified candidate”
- “production индекс”

Use:
- “Как быстро проверить подрядчика”
- “Чеклист перед договором”
- “Внешние источники”
- “Досье компании”
- “Адрес не подтверждён”
