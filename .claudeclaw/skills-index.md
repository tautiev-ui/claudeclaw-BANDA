# Скиллы команды BANDA

> При поступлении задачи - сверься с этим списком. Если подходящий скилл есть - используй.
> Скиллы лежат в `skills/` (проектные) и `~/.claude/skills/` (глобальные Claude Code)

## Встроенные скиллы (в проекте)

| Скилл | Когда использовать |
|-------|--------------------|
| **minimax-pdf** | Создание PDF: отчёты, документы, резюме |
| **minimax-docx** | Создание/редактирование Word документов |
| **minimax-xlsx** | Работа с Excel: таблицы, отчёты, данные |
| **pptx-generator** | Создание PowerPoint презентаций |
| **browser-use** | Автоматизация браузера через Playwright |
| **browser-use-api** | Cloud browser API для веб-задач |
| **gmail** | Отправка/чтение email через Gmail |
| **google-calendar** | Работа с Google Calendar |
| **slack** | Интеграция со Slack |
| **timezone** | Конвертация часовых поясов |
| **tldr** | Быстрые саммари длинных текстов |

## Рекомендуемые скиллы (установить в ~/.claude/skills/)

### Для всех агентов
| Скилл | Когда использовать |
|-------|--------------------|
| **deep-research-pro** | Глубокий ресёрч с веб-поиском, мультиисточники |
| **brainstorming** | ПЕРЕД любой творческой работой |
| **systematic-debugging** | Баг, ошибка, "не работает" - ПЕРЕД предложением фиксов |
| **quality-check** | Финальная проверка перед сдачей |
| **writing-plans** | "Составь план", "как реализовать" |
| **weather** | Погода в любом городе |
| **gemini** | Быстрый вопрос к Gemini (one-shot) |
| **audit-website** | SEO/перформанс аудит сайта |
| **youtube-seo** | SEO для YouTube: заголовки, описания, хештеги |
| **excalidraw** | Схемы и диаграммы |

### Для Гейла (программист)
| Скилл | Когда использовать |
|-------|--------------------|
| **test-runner** | Запуск тестов, парсинг результатов |
| **code-review** | Ревью кода, поиск антипаттернов |
| **dep-audit** | Аудит зависимостей: уязвимости, лицензии |
| **git-wizard** | Сложные git-операции: rebase, cherry-pick, конфликты |
| **perf-profiler** | Профилирование: Lighthouse, Node.js, SQL |

### Для Лало (маркетолог)
| Скилл | Когда использовать |
|-------|--------------------|
| **analytics** | Статистика YouTube + Telegram |
| **channel-analyzer** | Анализ Telegram каналов |
| **tweet-writer** | Написание постов для X/Twitter |
| **last30days** | Тренды за 30 дней в Reddit/X/Web |

## Decision Tree

```
Задача получена → определи тип:

├─ "Не работает / баг / ошибка"
│  └─ systematic-debugging
│
├─ "Напиши код / сделай фичу"
│  └─ writing-plans → [код] → test-runner → code-review → quality-check
│
├─ "Проверь / ревью код"
│  └─ code-review → quality-check
│
├─ "Тормозит / медленно"
│  └─ perf-profiler
│
├─ "Git / rebase / конфликт"
│  └─ git-wizard
│
├─ "Проверь сайт / SEO"
│  └─ audit-website
│
├─ "Напиши пост / контент"
│  └─ content-pipeline (Туко→Лало→Ким)
│
└─ "Схема / диаграмма"
   └─ excalidraw
```
