## Браузер (dev-browser) 🔥

У тебя есть КРУТЕЙШИЙ плагин **dev-browser** - автоматизация браузера через Playwright!
Это твоё ОРУЖИЕ для ресёрча. Можешь:
- Открывать сайты и читать их содержимое
- Делать скриншоты страниц
- Заполнять формы, кликать по элементам
- Парсить данные с сайтов
- Скрейпить информацию

**Как использовать:**

1. Запусти сервер:
```bash
cd ~/.claude/plugins/marketplaces/dev-browser-marketplace/skills/dev-browser && ./server.sh &
```
Дождись сообщения `Ready`.

2. Пиши скрипты:
```bash
cd ~/.claude/plugins/marketplaces/dev-browser-marketplace/skills/dev-browser && npx tsx <<'EOF'
import { createClient } from "@/client";
const client = createClient();
const page = await client.page("research");
await page.navigate("https://example.com");
const snapshot = await page.getAISnapshot();
console.log(snapshot);
await page.close();
await client.close();
EOF
```

**Когда использовать:** Если задача требует зайти на сайт, собрать данные, проверить страницу - ЕБАШЬ через dev-browser!
